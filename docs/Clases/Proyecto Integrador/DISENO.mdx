# Diseño

El juego se llama "Toca el botón" un juego, en donde esta hecho en impresora 3D, cuenta con 50 luces y un boton, entonces tiene sonido y luces, asi como una pantalla en donde te indica si estas jugadno o on.

Diseñé un juego inclusivo que combina múltiples estímulos sensoriales para garantizar la participación de niños con discapacidades visuales y auditivas, así como de sus familias. El dispositivo fue creado utilizando una impresora 3D y cuenta con una estructura resistente y ergonómica que lo hace adecuado para diferentes edades y habilidades.

El juego incluye las siguientes características:

1. **Luces LED** : Está equipado con 50 luces programadas para generar patrones dinámicos y atractivos, que proporcionan un estímulo visual entretenido y claro.
2. **Sistema de sonido** : Incorpora un parlante que emite sonidos interactivos, indicaciones y efectos, diseñados para guiar a los jugadores no videntes y enriquecer la experiencia.
3. **Pantalla indicadora** : Incluye una pantalla que informa al usuario sobre el estado del juego, como si está encendido o en modo de juego activo, lo que facilita su uso para cualquier persona.
4. **Interacción táctil** : Un botón central permite a los jugadores interactuar directamente con el dispositivo, facilitando un control intuitivo y accesible.

El diseño combina simplicidad y funcionalidad para que pueda ser utilizado sin dificultades por cualquier niño, independientemente de sus capacidades. Además, al emplear la impresión 3D, se logró una estructura personalizada que es ligera, segura y estética, ideal para un uso tanto doméstico como educativo.

Este proyecto busca demostrar que la tecnología puede ser una herramienta poderosa para crear productos accesibles, promoviendo la integración y el disfrute compartido en entornos familiares y sociales.

## Materiales y componentes utilizados:

- **Plástico PLA** : Material utilizado para la impresión 3D de la estructura del juego. Este material es fácil de imprimir, ecológico y adecuado para la creación de piezas duraderas y ligeras.
- **Luces LED** : Se usaron 50 luces LED para generar estímulos visuales en el juego, ofreciendo patrones de luz interactivos.
- **Parlante** : Un pequeño parlante integrado al sistema, diseñado para emitir sonidos, indicaciones y efectos relacionados con la interacción del juego.
- **Microcontrolador ESP32** : Utilizado para el procesamiento de las señales y la gestión de la interacción entre los diferentes componentes del juego, como las luces, el sonido y el pulsador de control.
- **Pantalla de leds** : Para mostrar el estado del juego, como si está encendido, en modo de espera o en juego activo, asi como animaciones.
- **Pulsador** : Permite la interacción directa del usuario con el juego, activando las luces y el sonido mediante su presión.
- **Cables y conectores** : Para la interconexión de los diferentes componentes electrónicos (luces, parlante, microcontrolador, etc.).
- **Batería recargable** : Fuente de alimentación que permite que el juego sea portátil y funcional sin necesidad de estar conectado a la corriente.

## Partes y sistemas fabricados:

- **Estructura del juego** : Fabricada mediante impresión 3D, esta parte incluye la carcasa principal y los compartimentos para los componentes electrónicos (microcontrolador, parlante, batería, etc.). La estructura está diseñada para ser fácil de montar y usar, con una interfaz simple.
- **Sistema de luces y sonido** : Las 50 luces LED están distribuidas a lo largo de la estructura del juego y conectadas al microcontrolador ESP32. El parlante se conecta igualmente al microcontrolador para emitir sonidos en función de las interacciones del usuario.
- **Pantalla de visualización** : Se incorporó una pantalla LCD o LED que indica el estado del juego, proporcionando retroalimentación visual a los usuarios.
- **Circuito de alimentación y control** : Incluye el sistema de distribución de energía desde la batería recargable hacia los componentes electrónicos, así como el circuito que gestiona la activación de las luces y el sonido según el uso del pulsador.

## Procesos utilizados:

- **Proceso Aditivo (Impresión 3D)** : Se utilizó una impresora 3D para fabricar la estructura principal del juego. La impresión 3D permitió crear piezas personalizadas con alta precisión, adaptadas a las necesidades específicas del diseño del juego.
- **Montaje Electrónico** : Se ensamblaron los componentes electrónicos, como las luces LED, el microcontrolador, el parlante y la pantalla, conectándolos de manera que interactúan según las señales del pulsador.
- **Programación y Configuración** : El microcontrolador ESP32 fue programado para gestionar las interacciones del usuario con el juego, controlando el encendido de las luces, la emisión de sonidos y la visualización en la pantalla, asegurando que todos los elementos funcionen de manera coordinada.
- **Proceso Sustractivo**: Se tuvieron que lijar piezas, asi como realizar pequeños cortes para lograr un acabado preciso y adecuado.

**Generación de los Sonidos**

Para la generación de los sonidos en el juego, se utilizó una voz generada mediante inteligencia artificial, lo que permitió ofrecer comandos y retroalimentación verbal de manera clara y precisa. Este enfoque permitió crear una experiencia auditiva personalizada, esencial para los usuarios no videntes. La voz de los comandos, como las instrucciones de juego o indicaciones de estado, fue generada utilizando **Evenlabs** , una plataforma de texto a voz (TTS) muy conocida por su calidad y precisión en la conversión de texto en voz natural.

Evenlabs proporciona una amplia gama de voces realistas y es altamente adaptable a las necesidades del proyecto, lo que permitió generar voces claras y fácilmente comprensibles para los usuarios. Gracias a esta herramienta, pude convertir los textos con los comandos y mensajes del juego en voces de alta calidad que enriquecen la experiencia sensorial.

Esta tecnología de TTS no solo hace que el juego sea interactivo en términos de luces, sino que también ofrece una dimensión auditiva que aumenta la accesibilidad para personas con discapacidad visual. La integración de Evenlabs permitió que el juego proporcionara una experiencia completamente accesible, no solo para los usuarios no videntes, sino también para sus familias y otros usuarios que puedan disfrutar de la interacción a través de sonidos.
